Fulcrum Query Utility

This lightweight, open source web application uses utilizes the Fulcrum Query API for fetching data from your Fulcrum account. Simply login to your Fulcrum account and begin writing SQL queries!

Features
  • Built on the extremely popular Bootstrap framework
  • Free form CodeMirror based SQL query editor
  • Queries saved to localStorage for convenience
  • Ability to save, delete, import, and export frequently used queries
  • Client-side table sorting, searching, and column toggling via Bootstrap Table
  • Export data to CSV & JSON
  • Share queries via URL `q` parameter: https://fulcrumapp.github.io/fulcrum-query-utility/?q=SELECT * FROM "My Table"

This lightweight, open source web application uses utilizes the Fulcrum Query API for fetching data from your Fulcrum account. Simply login to your Fulcrum account and begin writing SQL queries!

If you write a query that returns the record_id (aka fulcrum_id) then checkboxes will appear and you can delete data PERMANENTLY from your Fulcrum account.

This action cannot be undone so use caution.

This app is provided free and with no expressed warranty or support. Use at your own risk.
